Variáveis são palavras que tem um valor que pode variar associado a elas.

Exemplo:
```
nome vale "Safira"
numero vale 10
```

Neste exemplo de cima, a variável nome tem o valor de "gabriel", um txto e a variável numero tem o valor numérico de 10.


Você pode mostrar o valor da variável com o comando mostre, exemplo:
```
mostre nome
```
Nestea caso, o valor exido na tela será
```
Safira
```
Você pode unir os textos com as variáveis, exemplo:
```
mostre "Meu nome é ", nome
```
Neste caso, o valor exibido será 
```
Meu nome é Safira
```

Você também pode fazer as contas com as variáveis

```
mostre numero + 2
```
A saída será 
```
12
```
Isso por que a variável numero te m o valor de 10.

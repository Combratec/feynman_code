Mostrar algo na tela

Exibe alguma coisa na tela, um texto, um número ou o valor de uma variável.


# Declaramos uma variável
nome = "Safira"

# Mostra o valor da variável nome na tela
mostre nome

# Mostra o valor de um número
mostre 10

# Mostra vários tipos de variáveis diferentes
mostre "Nome: ", nome, " Idade: ", 25
Saída 
Safira
10.0
Nome: Safira Idade: 25.0

Se você notou, cada texto vem em uma linha, e se você quisesse manter tudo na mesma linha você pode usar o comando


# Declaramos uma variável
nome = "Safira"

# Mostra as variáveis em uma mesma linha
mostre nessa linha "Nome: "
mostre nessa linha nome
mostre nessa linha " Idade: "
mostre nessa linha 25
Saída 
Nome: Safira Idade: 25.0
